# 🚀 My React Landing Page for Prodesk IT

A modern, responsive landing page built using **React.js** and **Tailwind CSS**, featuring a clean UI, responsive navigation, and dark mode support.

---

## 📌 Features

* ⚡ Responsive Navbar (Mobile + Desktop)
* 🌙 Dark Mode Toggle
* 📱 Mobile-friendly Hamburger Menu
* 🎯 Hero Section
* 🧩 Services Section Layout
* 🔗 Footer with Social Media Icons
* 🎨 Smooth Hover Effects & Transitions

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** Tailwind CSS
* **Build Tool:** Vite

---

src/

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/Arjun8384/Mission1.git
```

### 2. Navigate to project folder

```
cd Mission1
```

### 3. Install dependencies

```
npm install
```

### 4. Run the development server

```
npm run dev
```

---

## 🎨 Tailwind Setup

Ensure Tailwind is configured properly:

### `index.css`

```css
@tailwind base;
@tailwind components;
@tailwind utilities;
```
